## 镜像

1. 列出本机所有 image 文件
   ```
   docker image ls
   ```
2. 删除 image 文件
   ```
   docker image rm [imageName]
   ```
3. 镜像拉取
   ```
   docker image pull hello-world
   ```


### **查看容器**

1. 列出当前所有正在运行的container
   ```
   docker ps
   ```
2. 列出所有container
   ```
   docker ps -a  
   ```
3. 列出最近一次启动的container

   ```
   docker ps -l  
   ```
